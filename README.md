+ Nguyễn Tuấn Anh.
+ Lê Thị Liên.
+ Lê Thu Hà. 
# How to play
Download Tetrix_code.exe in Tetrix_code folder.
Interace:
+ Using A to move block to the left.
+ Using D to move block to the right.
+ Using S to move block down faster.
+ Using W to rotate block.
+ Using esc to pause game.
